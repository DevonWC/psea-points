Banco de dados em sqlite

Tanto o operador (!) como o comando (pps) podem ser editados na configuração

Comandos:
  - !pps : Ver todos os pontos
  - !pps @nome : ver os pontos da pessoa mencionada
  - !pps @nome 123 : adicionar ou remover pontos da pessoa mencionada
     - !pps @nome -1 : Remove 1 ponto
     - !pps @nome 1 : Adiciona 1 ponto
